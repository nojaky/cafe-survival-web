# 카페 창업 생존 시뮬레이터 Web

파이썬 Tkinter 버전의 `카페 창업 생존 시뮬레이터`를 GitHub Pages에서 실행할 수 있는 정적 웹앱으로 변환한 버전입니다.

## 구성

- `index.html`: 화면 구조
- `styles.css`: 반응형 UI 스타일
- `app.js`: 계산 로직, 시나리오 비교, 위험 진단, 차트, JSON 저장/불러오기

## 배포

GitHub 저장소에 파일을 업로드한 뒤 Pages를 켜면 바로 사용할 수 있습니다.

## 변환된 기능

- 초기 자금, 매출 조건, 변동비, 고정비 입력
- 1인 개인카페, 일반 개인카페, 저가 프랜차이즈 예시 불러오기
- 예상 월매출, 월 영업이익, 손익분기 매출, 운영자금 생존기간 계산
- 24개월 예상 현금흐름 캔버스 차트
- 낙관적, 기준, 보수적, 위기 시나리오 비교
- 안전점수와 위험요인 진단
- JSON 저장 및 불러오기
- 네이버 톡톡 문의 버튼
